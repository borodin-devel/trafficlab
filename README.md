# Trafficlab

Первый черновик проекта для воспроизводимого конвейера: захват, преобразование,
проверка, обучение, генерация, сравнение и эволюция моделей трафика.

Архитектура и связи между стадиями описаны в [architecture/README.md](architecture/README.md).

Запуск CLI после установки: `trafficlab --help`.
