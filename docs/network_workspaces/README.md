# Network Workspace Designs

This directory elaborates the network-workspace alternatives for the
[capture architecture](../../architecture/apps/10_capture/README.md).

Architecture documents own the decisions; these documents explain their
implementation trade-offs and boundaries.

[development environment](../../architecture/DEVELOPMENT.md).
