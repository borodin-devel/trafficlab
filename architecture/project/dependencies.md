# Python и зависимости

## Версия Python

Проект использует Python 3.12. Границы версии фиксируются одновременно в `pyproject.toml` и `.python-version`:

```toml
requires-python = ">=3.12,<3.13"
```

```text
3.12
```

Ubuntu 24.04 поставляет Python 3.12 как системную ветку. Отдельная поддержка Python 3.11 не требуется. Ветка 3.12 получает исправления безопасности до октября 2028 года.

## Управление окружением

`uv` управляет версией Python, виртуальным окружением, зависимостями и lock-файлом `uv.lock`. Воспроизводимая проверка использует зафиксированное разрешение:

```text
uv sync --frozen
uv run pytest
uv run pyright
uv run ruff check .
uv run ruff format --check .
```

Первая версия является одним проектом. Workspace `uv` не используется, чтобы не создавать лишние границы упаковки.

## Опциональные группы

```text
ml-torch:
    torch
    safetensors

evolution:
    pymoo или DEAP

dev:
    pytest
    pytest-cov
    pytest-xdist
    pytest-timeout
    hypothesis
    pyright
    ruff
```

Группа `ml-torch` нужна только нейронным моделям. Группа `evolution` добавляет библиотеку многокритериальной оптимизации. Группа `dev` содержит тестирование, покрытие, статическую проверку типов и форматирование.

Конкретный набор и версии определяет `uv.lock`. Добавление обязательной зависимости требует объяснить, какой модуль её использует, почему стандартной библиотеки недостаточно и можно ли оставить зависимость опциональной.
