# ON/OFF Pareto Software Architecture Document

## Context, Goals, and Boundaries

Goal is future heavy-tailed burst/idle modelling. Current documentation does
not define Pareto convention and support, initial phase, ON-period packet law,
frame lengths, fitting/censoring, finite-mean constraints, stops, or schema.

## Architecture Status and Risks

Selection remains unsupported. Research must address infinite-mean parameter
regions, truncation/censoring, boundaries between periods, zero-length periods,
deterministic sampling, resource bounds, and statistical correctness tests.
