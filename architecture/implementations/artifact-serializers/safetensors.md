# Safetensors

## Назначение

Сериализатор Safetensors сохраняет веса нейронных сетей по [контракту артефакта модели](../../contracts/model-artifact.md). Он используется моделями, результат обучения которых представлен тензорами весов.

## Применение

Реализация записывает веса в файл Safetensors и передаёт сведения о созданном файле издателю артефакта. Метаданные и другие части результата используют предусмотренные контрактом форматы.

## Ограничения

Safetensors предназначен для весов нейронной сети. Допустимые файлы, их размещение и правила проверки определяет контракт артефакта модели.
