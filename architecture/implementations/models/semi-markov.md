# Полумарковская модель

## Назначение

Модель `semi_markov` — рекомендуемая первая содержательная модель. Полумарковская модель описывает переходы между состояниями и отдельно задаёт длительность пребывания до следующего события.

## Входные признаки

Состояние включает направление, класс протокола, класс всплеска трафика и состояние потока. Наблюдаемое событие содержит временной интервал, длину и флаги. Требуемые поля и их типы реализация объявляет как feature schema — схему входных признаков.

## Обучение

Transition model — модель переходов — оценивает выбор следующего состояния. Duration model — модель длительности — оценивает время до события. Emission model — модель наблюдений — оценивает длину и флаги события.

## Генерация

Для текущего состояния модель последовательно выбирает следующее состояние, время до события, длину и флаги. Исходное значение генератора псевдослучайных чисел (seed) задаёт воспроизводимую последовательность выборок. Модель переходов позволяет ограничивать допустимые переходы между протоколами.

## Отличия и ограничения

Модель работает на центральном процессоре — Central Processing Unit (CPU), интерпретируема и не требует большого набора данных. Её параметры удобно изменять в эволюционном цикле. Качество результата зависит от выбранного определения состояния, включая классы протокола и всплеска.
