# Авторегрессионная модель Transformer

## Назначение

Модель `transformer_autoregressive` использует ограниченный Transformer для последовательного описания событий трафика. Авторегрессионная генерация использует контекст предыдущих событий для выбора следующего.

## Входные признаки

Модель получает окна последовательностей событий. Точный состав признаков и их типы реализация объявляет как feature schema — схему входных признаков.

## Обучение

Transformer обучается связывать ограниченное окно контекста со следующим событием. Реализация использует небольшую embedding dimension — размерность векторного представления — и небольшое context window — окно контекста.

## Генерация

Модель авторегрессионно добавляет события, используя доступное окно предыдущих событий. Исходное значение генератора псевдослучайных чисел (seed) задаёт воспроизводимый выбор при генерации.

## Ограничения

Обучение модели ресурсоёмко и подчиняется [общей политике ресурсов](../../common/resources.md). Точные размерность представления, длину окна и состав признаков задаёт конфигурация.
