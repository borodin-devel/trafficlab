# Условная эмпирическая модель

## Назначение

Модель `conditional_empirical` сохраняет выбранные зависимости между признаками, используя условные эмпирические распределения.

## Входные признаки

Модель использует длину пакета, направление, предыдущее направление, протокол, inter-arrival time — интервал между соседними событиями, флаги Transmission Control Protocol (TCP) и состояние потока. Требуемые поля и их типы реализация объявляет как feature schema — схему входных признаков.

## Обучение

Реализация оценивает по наблюдениям следующие распределения:

```text
P(length | direction, protocol)
P(delta | direction, previous_direction, protocol)
P(flags | protocol, flow_state)
```

Здесь `P` обозначает условное распределение вероятности, `delta` — inter-arrival time, а `flow_state` — состояние потока.

## Генерация

Модель выбирает длину, временной интервал и флаги из распределений, соответствующих текущим условиям. Исходное значение генератора псевдослучайных чисел (seed) задаёт воспроизводимую последовательность выборок.

## Ограничения

Модель описывает только перечисленные условные зависимости. Она не вводит отдельную модель переходов между состояниями и не оценивает зависимости вне выбранных условий.
