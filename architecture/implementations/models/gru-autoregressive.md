# Авторегрессионная модель GRU

## Назначение

Модель `gru_autoregressive` использует Gated Recurrent Unit (GRU) — рекуррентный блок с управляемыми вентилями — для последовательного описания событий трафика. Авторегрессионная генерация использует уже полученную часть последовательности для выбора следующего события.

## Входные признаки

Модель получает последовательности событий, разделённые по сессиям. Точный состав признаков и их типы реализация объявляет как feature schema — схему входных признаков.

## Обучение

GRU обучается на последовательностях событий из достаточного числа сессий. Обучение связывает предшествующую часть последовательности со следующим событием.

## Генерация

Модель начинает с заданного контекста и авторегрессионно добавляет по одному событию. Исходное значение генератора псевдослучайных чисел (seed) задаёт воспроизводимый выбор при генерации.

## Ограничения

Модель следует использовать после появления достаточного числа сессий. Архитектура не задаёт минимальное число сессий, состав признаков или параметры обучения: их объявляет конкретная конфигурация вместе с feature schema.
