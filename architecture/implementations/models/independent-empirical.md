# Независимая эмпирическая модель

## Назначение

Модель `independent_empirical` служит минимальной базовой моделью — baseline и точкой отсчёта для сравнения. Она также проверяет полный конвейер обучения и генерации.

## Входные признаки

Модель использует длину пакета, inter-arrival time — интервал между соседними событиями, направление, протокол и флаги Transmission Control Protocol (TCP). Требуемые поля и их типы реализация объявляет как feature schema — схему входных признаков.

## Обучение

Реализация строит отдельное эмпирическое распределение каждого признака по наблюдаемым данным. Совместные зависимости между признаками не оцениваются.

## Генерация

Каждый признак семплируется независимо из своего эмпирического распределения. Исходное значение генератора псевдослучайных чисел (seed) задаёт воспроизводимую последовательность выборок.

## Ограничения

Модель не сохраняет условные связи между признаками и последовательную структуру трафика. Поэтому она предназначена для базовой проверки, а не для описания зависимостей между событиями.
