# Последовательностная модель VAE

## Назначение

Модель `vae_sequence` использует Variational Autoencoder (VAE) — вариационный автокодировщик — для последовательностей событий трафика.

## Входные признаки

Модель получает последовательности событий. Точный состав признаков и их типы реализация объявляет как feature schema — схему входных признаков.

## Обучение

VAE обучает кодирующую и декодирующую части на последовательностях событий. Архитектура не фиксирует внутреннее устройство этих частей или параметры обучения.

## Генерация

Декодирующая часть формирует последовательность из выбранного скрытого представления. Исходное значение генератора псевдослучайных чисел (seed) задаёт воспроизводимый выбор при генерации.

## Ограничения

Модель не входит в минимальный рабочий продукт. Её архитектура, состав признаков и параметры обучения должны быть определены до реализации.
