# Wavelet-Scaling Similarity — Unresolved

## Purpose and Status

This planned method would compare burstiness/scaling across time scales with a
wavelet representation. It is a research stub, not registered or selectable.

## Inputs, Outputs, and Dependencies

Series, wavelet, boundary mode, levels, coefficients/statistics, distance,
weights, score, and numerical library are unresolved.

method offline and unprivileged. [PCAP I/O](../../libs/pcap_io/README.md) owns
input mechanics. The
[similarity-result contract](../../contracts/60_30_similarity_result/README.md) owns
publication.
Unresolved methods remain unselectable.
