# Joint Sinkhorn/Wasserstein Configuration

Required finite positive fields are IAT feature scale, frame-length feature
scale, entropic regularization, score mapping scale, absolute/relative solver
tolerance, and positive maximum iterations. Implementation additionally fixes
solver, backend, algorithm, seed, threads/device, and deterministic controls and
records them. Exact TOML path/default values are unresolved; no silent defaults
may be implemented before schema approval.
